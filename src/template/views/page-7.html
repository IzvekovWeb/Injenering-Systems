Центробежные насосы для нефтяной, нефтехимической и газовой промышленности (API Standard 610 / ISO 13709)

В нефтяной, нефтехимической и газовой отраслях требуется применение сертифицированного насосного оборудования,
которое соответствует международным стандартам.


Стандарт API 610 соответствует международному стандарту ISO 13709. Данный стандарт описывает требования к насосному оборудованию и его рабочим характеристикам. Основной областью применения данных насосов является нефте-
и газопереработка, химическая промышленность и объекты нефтедобычи.


Достоинство насосов по стандарту API 610 состоит в том, что предусматривает максимальный срок службы насосного оборудования — не менее 20 лет при межремонтном ресурсе 3 года. Корпус насоса способен выдерживать огромные нагрузки
при температурах не менее 150 градусов и при давлении до 600 бар. При этом плотность перекачиваемых жидкостей может быть очень высокой.


Стандарт API 610 может быть применен к различным насосам: консольным, вертикальным, двухопорным, полупогружным, одно- или многоступенчатым, двухкорпусным. В качестве материала корпуса и проточной части используются чугун, хромистая, углеродистая, нержавеющая сталь,а также различные сплавы под любые задачи и требования заказчиков.

Типы насосов по стандарту API 610:

Двухопорный насос с осевым разъемом корпуса Тип ВВ-1 Двухопорный многоступенчатый насос с горизонтальным осевым разъемом корпуса Тип ВВ-3
Двухопорный насос с радиальным разъемом корпуса Тип ВВ-2 Одноступенчатый насос консольного типа Тип ОН-2
Центробежный многоступенчатый двухкорпусной насос Тип ВВ5

Одноступенчатый насос консольного типа Тип ОН-2


Одноступенчатый консольный с радиальным разъемом корпуса насос
разработан для транспортировки технической воды, углеводородов,
продуктов химии и технологических продуктов различного применения
на НПЗ, предприятиях химической и нефтехимической отраслей.

Преимущества
Возможность быстрого запуска и остановки;
Высокая точность регулировки подачи перекачиваемой среды;
Длительный межремонтный цикл работы с назначенным сроком службы 20 лет;
Высокая ремонтопригодность, легкость в техническом обслуживании;
Возможность перекачивать вызрывопожароопасные и химически агрессивные среды;
Высокий КПД;
Широкий диапазон работы насоса до 2850 м3/ч с напором до 350 м
Работа на высоких температурах до 425 °С

Двухопорный насос с осевым разъемом корпуса Тип ВВ-1


Рабочее колесо двустороннего входа. Корпус выполнен в виде двойной спирали, имеет крепление на оси, в двух подшипниковых узлах. Насос был специально разработан для работы в трубопроводных системах, но также существует версия, полностью соответствующая требованиям стандарта API 610. Насос предназначен для работы в трубопроводных системах, системах пожаротушения, а такжедля создания повышенного давления. Насос сконструирован с учетом требований максимальной надежности и простоты эксплуатации.

Преимущества
Насосы выполнены в полном соответствии со стандартом API 610
Осевой баланс увеличивает срок службы, снижает эксплуатационные затраты
Высокая производительность до 15 000 м3/ч с напором до 250 м
Жесткая и прочная конструкция
Полная стандартизация для широкой сферы применения

Двухопорный насос
с радиальным разъемом корпуса Тип ВВ-2


Горизонтальный двухступенчатый насос с радиальным разъемом корпуса, двухопорные. Насос предназначен для работы в условиях высоких давлений. Фланцы ориентированы вверх. Насос монтируются по центральной оси, предназначен для тяжелых условий эксплуатации, исполнение полностью соответствует стандарту ISO 13709 – API 610. Для понижения значений требуемого кавитационного запаса насос может выполняться с рабочими колесами первой ступени двустороннего входа.

Преимущества
Насосы выполнены в полном соответствии со стандартом API 610
Осевой баланс увеличивает срок службы, снижает эксплуатационные затраты
Высокая производительность до 2500 м3/ч с напором до 750 м
Рабочее давление до 80 бар
Высокие рабочие температуры до 450 °С
Жесткая и прочная конструкция
Полная стандартизация для широкой сферы применения

Двухопорный многоступенчатый насос с горизонтальным осевым разъемом корпуса Тип ВВ-3


Многоступенчатый насос с горизонтальным разъемом корпуса, однокорпусной, для высоких давлений, с рабочими колесами, расположенными симметрично. Насос выполнен в соответствии с требованиями стандарта API 610. Доступны различные комбинации размеров и материальных исполнений, пригодные для использования в широком диапазоне процессов и промышленных применений. Ряд данного типа насосов разработан для непрерывной использования
во всех процессах нефтепереработки, нефтехимической, химической
и основной промышленности. Насосы предназначены для тяжелых условий работы.

Преимущества
Высокая производительность, жесткая и прочная конструкция
Широкой диапазон производительности от 10 – 2800 м3/ч с напором до 2500 м
Рабочее давление до 275 бар
Симметричное расположение рабочих колес для осевого баланса
Статически и динамически сбалансированные рабочие колеса

Центробежный многоступенчатый двухкорпусной насос Тип ВВ5


Многоступенчатый центробежный насос двухкорпусный, с радиальным торцевым разъемом предназначен для перекачивания чистых
или слегка загрязненных сред, а также химически нейтральных
и коррозионных жидкостей, а также в многочисленных процессах
как первичной, так и вторичной переработки нефти, и изготавливаемых
из нее нефтепродуктов.

Преимущества
Высокая производительность, жесткая и прочная конструкция
Широкой диапазон производительности от 12,5 до 600 м3/ч с напором до 4000 м
Рабочее давление до 40 бар
Температура жидкости, перекачиваемой насосом: от -45 до + 450 °С
Возможность перекачивать высоковязкие жидкости до 300 сСт
Максимальная скорость ротора до 6000 об/мин
